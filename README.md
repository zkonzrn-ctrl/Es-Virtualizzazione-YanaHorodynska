Breve descrizione : 
Obiettivo: Creare una presentazione coinvolgente con Prezi che spieghi i concetti di virtualizzazione e l'importanza di gestire il codice con GitHub.


Area 1: Virtualizzazione
Ipervisore: Software che fa girare più sistemi operativi sullo stesso hardware fisico.
VirtualBox: Ipervisore di Tipo 2 (Hosted). Si installa come un normale programma su Windows o Linux.
Vantaggi: Isolamento (sicurezza), Test (esperimenti senza rischi) e Risparmio hardware (meno macchine fisiche).

Area 2: GitHub
Cos'è: Piattaforma cloud basata su Git per gestire codice e collaborare.
Backup: Il lavoro è al sicuro online, fuori dalla macchina virtuale.
Versionamento: Storia delle modifiche tramite i Commit per non perdere nulla.
Portfolio: Il tuo curriculum digitale per mostrare i tuoi progetti reali.
